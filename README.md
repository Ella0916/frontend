### HTML Elements
> https://www.w3schools.com/html/html_elements.asp

`(backtick)
```
<tagname> 첫번째 수업 </tagname>
* 시작태그만 있는 경우 : Empty Element 
```

> 포함관계(nested)
```
<html>
  <body>
    <h1>Heading</h1>
  </body>
</html>
```

> html-body-h1 관계<p>
>> html : body와의 관계 - 부모요소 / h1과의 관계 - 조상요소<p>
>> body : html과의 관계 - 자식요소 / h1과의 관계 - 자식요소<p>
>> h1 : body와의 관계 - 자식요소 / html과의 관계 - 자손요소
