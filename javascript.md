## Javascript 변수 : var

https://www.w3schools.com/js/js_variables.asp

- 변수를 정의/ 선언할 때 사용하는 키워드(예약어) 이다.
- ES5 버전에서 사용

```
var a;
var b;

var a, b; > 로 한번에 여러개를 선언할 수 있다.
```

## Javascript 변수 : let

https://www.w3schools.com/js/js_let.asp

- 변수를 정의/ 선언할 때 사용하는 키워드(예약어) 이다.
